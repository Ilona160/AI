# 💡 Практичні кейси використання ШІ

На цій сторінці зібрані реальні приклади того, як штучний інтелект допомагає у навчанні, творчості та розв'язанні побутових завдань.

---

## 📝 Кейс 1: Покращення тексту (Промпт-інженерія)

**Завдання:** Учень написав коротке есе, але воно звучить занадто просто. Потрібно зробити його більш науковим та структурованим.

> **Початковий текст:** "ШІ — це круто. Він допомагає нам робити уроки і малювати картинки. Скоро він буде всюди."

> **Промпт для ШІ:** "Ти — професійний редактор. Перетвори цей текст на науковий абзац для шкільної доповіді. Використовуй терміни: автоматизація, генеративні моделі, інтеграція."

**Результат ШІ:**
> "Інтеграція штучного інтелекту в повсякденне життя трансформує освітній процес. Сучасні генеративні моделі не лише оптимізують виконання навчальних завдань, а й відкривають нові горизонти для творчості, забезпечуючи автоматизацію складних інтелектуальних процесів."

> **Промпт для Midjourney/DALL-E:** "Futuristic eco-city, high-tech buildings covered in vertical gardens, flying electric vehicles, clean blue river, cinematic lighting, 8k resolution, hyper-realistic."

## 💻 Кейс 3: Допомога у програмуванні (Python)

**Завдання:** Написати код, який вираховує середній бал учня.

**Запит до ШІ:** "Напиши на Python програму, яка запитує у користувача 5 оцінок і виводить їх середнє арифметичне. Додай коментарі до кожного рядка."

**Код від ШІ:**
```python
# Створюємо список для оцінок
grades = []

# Цикл для введення 5 оцінок
for i in range(5):
    grade = float(input(f"Введіть оцінку №{i+1}: "))
    grades.append(grade)

# Рахуємо середнє
average = sum(grades) / len(grades)

print(f"Ваш середній бал: {average}")
