Теорія Штучного Інтелекту

Блок 1. Природа та основи ШІ

Що таке ШІ та його класифікація

Штучний інтелект (ШІ) — це не одна програма, а ціла галузь, що створює системи, здатні вчитися на досвіді. Ми розрізняємо:

  • Вузький ШІ (ANI): Майстер однієї справи (наприклад, розпізнавання пісні в Shazam).
  • Загальний ШІ (AGI): Рівень людського мозку (поки лише в теорії).

Машинне навчання (ML)

Це спосіб "вирощування" інтелекту. Замість коду ми даємо дані. Основні методи:

  • Навчання з учителем (на розмічених прикладах).
  • Навчання без учителя (пошук закономірностей).
  • Навчання з підкріпленням (метод батога та пряника).

Блок 2. Як працюють нейромережі та LLM

Нейронні мережі імітують біологічні нейрони. Вони складаються з шарів, де кожен шар "витягує" певні ознаки з даних.

"Сучасні мовні моделі (LLM), як-от GPT, не розуміють текст так, як ми. Вони прогнозують ймовірність появи наступного слова."

Промпт-інженерія

Для отримання якісної відповіді використовуйте структуру: Роль + Контекст + Завдання + Формат.

Блок 3. Генеративні технології

Генеративний ШІ створює новий контент: тексти, зображення, музику та код. Для зображень використовується дифузія — процес перетворення цифрового шуму на чітку картинку за запитом користувача.

Блок 4. Етика та Майбутнє

Головні виклики сучасності:

  • Упередженість: ШІ може копіювати людські стереотипи.
  • Дипфейки: Ризик створення фальшивих відео та аудіо.
  • Академічна доброчесність: Використання ШІ для допомоги, а не для заміни власного мислення.
